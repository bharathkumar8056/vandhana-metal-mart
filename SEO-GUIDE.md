## 🚀 Next Steps After Domain Purchase

### Step 1: Update URLs
After buying your domain, update these files:

**File: `app/layout.tsx`**
```typescript
metadataBase: new URL('https://your-domain.com'),
```

**File: `public/robots.txt`**
```
Sitemap: https://your-domain.com/sitemap.xml
```

**File: `public/sitemap.xml`**
Replace all instances of:
```
https://vandhanaa-metal-mart.netlify.app
```
with:
```
https://your-domain.com
```

### Step 2: Google Search Console Setup
1. Go to [Google Search Console](https://search.google.com/search-console)
2. Add your property (domain)
3. Verify ownership using one of these methods:
   - HTML file upload
   - DNS verification (recommended)
   - Google Analytics
4. Submit your sitemap: `https://your-domain.com/sitemap.xml`

### Step 3: Google Business Profile
1. Create/claim your [Google Business Profile](https://business.google.com)
2. Add complete business information:
